Cineflix

A fullstack Netflix clone, built on a Ruby on Rails backend, with a React/Redux frontend. Users may browse the entire collection of Cineflix movie trailers/clips and save them to a personal collection. Users can also remove movies from such collection, as well as search for a specific movie by title, in a dynamic searchbar. Movies are classified by genre, and can be browsed by genre as well.

Cineflix incorporates custom user authentication to securely log users in/out, including relevant authentication error messages. Amazon Web Services (S3) is utilized for server-side storage for all multimedia files.

Pandamonium

Pandamonium is a fullstack social media app created to foster an inclusive and welcoming environment for the LGBTQ community and allies. Pandamonium was built on a MongoDB backend, also implementing Mongoose ODM to map data from the backend as Javascript objects. React/Redux was utilized for the frontend. The app also implements Google maps API to identify city/state locations to create geographic forums.

Account holders can create a profile linked to a certain regional forum. All users in a given forum can make posts to that forum as well as sub-posts. Posts can also be downvoted/upvoted by users. Relatedly, there is also a built-in 'anonymous' function to help users communicate anonymously and ease into their new community. User profiles can be customized with a short bio and a unique avatar.
